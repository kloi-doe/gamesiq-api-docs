## Endpoints
### `GET /assets/`
assets_list
```bash
curl -X GET "https://YOUR-API-HOST/assets/" \
  -H "Authorization: Bearer <TOKEN>"
```
### `POST /assets/`
assets_create
```bash
curl -X POST "https://YOUR-API-HOST/assets/" \
  -H "Authorization: Bearer <TOKEN>"
```
### `DELETE /assets/{id}/`
assets_destroy
```bash
curl -X DELETE "https://YOUR-API-HOST/assets/<id>/" \
  -H "Authorization: Bearer <TOKEN>"
```
