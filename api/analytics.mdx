## Endpoints
### `GET /analytics/top_games_categories_list/`
Get Top Game Categories
```bash
curl -X GET "https://YOUR-API-HOST/analytics/top_games_categories_list/" \
  -H "Authorization: Bearer <TOKEN>"
```
### `GET /analytics/top_games_list/`
Get Top Games List
```bash
curl -X GET "https://YOUR-API-HOST/analytics/top_games_list/" \
  -H "Authorization: Bearer <TOKEN>"
```
